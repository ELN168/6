【好消息】現在就能上線，什麼都不用設定。
          社群連結已經幫你填好了。


--- 你只要做這一件事 --------------------------------------------

   把 index.html 和 config.js 這 2 個檔案上傳到 GitHub。
   （assets 資料夾的 24 張照片，你 GitHub 上本來就有，不用重傳）

   手機做法：
     1. 打開 github.com/wsad952701-beep/123
     2. 進 13 資料夾
     3. 右上角 → Add file → Upload files
     4. 把 index.html 和 config.js 丟進去（會自動覆蓋舊的）
     5. Commit changes
     6. 等 2 分鐘 → 完成！


--- 客人會看到什麼 ----------------------------------------------

   打開網站 → 自由看菜單
        |
        |  按「＋」想點餐
        v
   跳出「訂餐前，請先加入順記 LINE 社群」
        |
        |  按綠色的「加入 LINE 社群」
        v
   手機直接喚起 LINE App → 進入你的社群
        |
        |  切回瀏覽器
        v
   ✓ 已加入順記社群 → [ 開始點餐 ]
   （剛才想點的那道，自動幫他加好）
        |
        v
   點餐單裡按「＋ 加餐點」
   → 跳出菜單：品名、價格、＋ − 調份數
        |
        v
   填資料 → 送出訂單
        |
        v
   ① 訂單自動複製
   ② 一鍵開社群
   ③ 長按輸入框 → 貼上 → 送出


   ★ 頂部那顆 LINE 按鈕 → 點了直接開你的社群，不繞路。
   ★ 沒加入社群，就點不了餐。


--- 訂單為什麼要客人自己貼？ ------------------------------------

   LINE 官方規定：社群（OpenChat）不支援網頁直接發訊息。
   全世界的網站都做不到，不是我們的問題。

   所以最可靠的方式就是「自動複製 → 開社群 → 貼上」，約 10 秒。
   送出後畫面會有三步驟圖和綠色大按鈕，客人照做就好。

   點餐單不會提早清空，一定要客人按「我已經貼上了」才清，
   免得他還沒貼上，單子就不見了。


--- 想更進階？（選配，不做也沒關係）-----------------------------

   去 LINE 拿一組 LIFF ID 填進 config.js，
   客人加入社群後會再跳一次「LINE 官方授權畫面」，
   按「允許」才能點餐 —— 你就能拿到他真實的 LINE 名稱。

   ★ 不需要 LINE 官方帳號，用你自己平常的 LINE 帳號就好，免費。

   詳細步驟看 SETUP.md。
