# 順記燒臘館 v9 升級說明

## 🆕 新增功能

### 1. 📋 獨立後台管理頁（admin.html）
- 網址：`https://你的網址/admin.html`
- 帳號密碼同 admin.js 裡設定的
- 五大管理功能（見下方）

### 2. 🔥 今日特餐
- 後台可挑選任何品項設為「今日特餐」
- 可設定特惠價（會自動劃掉原價）
- 前台首頁會出現醒目的「今日特餐」區塊
- 客人一進頁面就看到

### 3. 📦 數量管理（剩餘份數）
- 後台可設定每道菜今日剩幾份
- 前台會顯示「剩 N 份」標籤
  - 綠色 = 充足（>5份）
  - 黃色 = 快賣完（≤5份）
  - 紅色 = 售完（0份）
- 數量歸 0 自動標售完

### 4. 📢 公告橫幅
- 後台輸入公告文字 → 前台頂部出現橫幅
- 三種樣式：一般（綠）/ 重要（黃）/ 促銷（金）
- 留空 = 不顯示

### 5. 📡 GitHub 一鍵發佈
- 後台設定好 GitHub Token 後
- 改完價格/特餐/數量 → 按「一鍵發佈」
- 自動更新 menu-data.js → 客人 2 分鐘內看到
- 不用再手動複製貼上！

---

## 🚀 怎麼用

### 方法 A：一鍵發佈（推薦）

1. 打開 `admin.html` → 登入
2. 到「發佈設定」頁籤 → 填入 GitHub 資料：
   - **Owner**：你的 GitHub 帳號
   - **Repo**：你的倉庫名稱
   - **Path**：`menu-data.js` 的路徑（例：`13/menu-data.js` 或 `menu-data.js`）
   - **Token**：到 github.com/settings/tokens → Generate → 勾 repo → 複製
3. 按「儲存設定」（只要設一次）
4. 以後每次改完 → 按底部「一鍵發佈」就搞定

### 方法 B：手動發佈（跟舊版一樣）

1. 在 `admin.html` 或網站 `#admin` 改完
2. 到「發佈設定」→ 複製設定碼
3. 到 GitHub 打開 `menu-data.js` → 全選貼上 → Commit

## 📁 檔案清單

| 檔案 | 說明 |
|------|------|
| `index.html` | 前台（已加入公告/特餐/數量顯示） |
| `admin.html` | **🆕 獨立後台管理頁** |
| `admin.js` | 後台邏輯（已升級支援新欄位） |
| `menu-data.js` | 菜單設定資料（已升級格式） |
| `config.js` | 基本設定（不用改） |

---

## ⚠️ 注意事項

- `admin.html` 的帳號密碼跟 `#admin` 共用，都在 `admin.js` 最上面設定
- GitHub Token 存在你自己手機的瀏覽器裡，不會被別人看到
- 公告、特餐、數量的修改，都是透過 `menu-data.js` 傳遞給前台
- 舊版的 `#admin` 入口照舊可用，不衝突
