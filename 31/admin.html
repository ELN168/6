直接改數字就好。🚫 = 今日售完（客人看到但不能點）　👁 = 下架（整個藏起來）

從菜單選品項加入「今日特餐」，可以設特惠價。前台會出現醒目的特餐區塊，客人一進來就看到。

設定今日剩餘份數。客人會看到「剩 N 份」的提示。數量歸 0 時自動顯示售完。
留空 = 不顯示數量（預設）。

輸入公告內容，會顯示在前台最上方。留空 = 不顯示。
例：「明天（週六）公休」「本週五加賣魚蛋、牛腩」

怎麼拿 Token？
① 到 github.com/settings/tokens
② Generate new token → Classic
③ Note 填「順記後台」、Expiration 選 No expiration
④ 勾選 repo（第一個）→ Generate
⑤ 複製 token 貼到下面
